# Site — Gases Perfeitos

Projeto escolar em formato de apresentação horizontal/canvas, com layout inspirado em Material Design 3 + estética limpa/minimalista.

## Como usar

1. Extraia o arquivo `.zip`.
2. Abra o arquivo `index.html` no navegador.
3. Use as setas do teclado, os botões inferiores ou arraste para o lado no celular.

## Recursos incluídos

- Navegação horizontal sem scroll vertical tradicional.
- Responsivo para celular e computador.
- Modo apresentação com tela cheia.
- Visão geral dos slides.
- Barra de progresso.
- Animações suaves.
- Mini laboratório interativo na transformação isotérmica.
- Imagens/ilustrações próprias em SVG.
- Ilustração do quadro usada como base do conteúdo.
- Funciona offline, sem CDN e sem bibliotecas externas.
